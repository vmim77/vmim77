[ 지도 API 넣기 ]

카카오 API 키 발급받기 ==> Javascript : 8944906c0a5b6e10fd2c53f0e414f6ce

마커를 찍어주고 마커에 마우스를 올리면 말풍선(윈도우)가 나온다.

다른 마커에 마우스를 올려주면 자동으로 이전에 말풍선(윈도우)는 사라진다.

지도 위에 마우스를 클릭하면 위, 경도 좌표값을 보여준다.

[ 매장 찾기 ] 메뉴 클릭시 지도의 위치 기본값은 사용자가 접속한 현재 위치로 한다.
==> 핸드폰은 위성을 이용해 신호를 송출하기 때문에 자세한 위치가 나온다.
==> PC나 노트북은 대략적인 위치가 나온다.(IP주소를 발급해주는 곳의 위치)

★ 지도는 Ajax인데 반드시 동기통신으로 해야한다.


▶ 매장찾기(카카오지도) 테이블 생성하기
- 매장ID
- 매장명
- 매장 홈페이지주소
- 매장소개 이미지파일명
- 매장주소 및 매장전화번호
- 위도
- 경도
- zindex
